## Typography

### Primary Font
- **Poppins** (Display)
  - Headings, logos, important text

### Secondary Font
- **Inter** (Body)
  - Paragraphs, buttons, general text

## Logo Usage Guidelines

### Clear Space
Maintain minimum clear space equal to the height of the "B" in Bluevision around the logo.

### Minimum Size
- Print: 1.5cm width
- Digital: 150px width
- Social Media: 400px width

### Background Colors
- Use primary logo on light backgrounds
- Use white logo on dark backgrounds
- Ensure sufficient contrast (minimum 4.5:1)

## Social Media Sizes

### Cover Photos
